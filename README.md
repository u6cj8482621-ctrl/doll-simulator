# Paper Doll Simulator

這是一個單頁版紙娃娃/娃包模擬器，目前主程式在 `index.html`。

## 怎麼打開

直接用瀏覽器打開 `index.html`。

如果畫面看得到控制面板但中間圖片破圖，代表程式本身有跑起來，只是圖片素材還沒放進對應資料夾。

## GitHub 網頁版上傳方式

1. 打開 GitHub 網站。
2. 右上角按 `+`，選 `New repository`。
3. Repository name 可填 `paper-doll-simulator`。
4. 先不要勾 `Add a README file`。
5. 建好後按 `Add file`，選 `Upload files`。
6. 把這個資料夾裡的 `index.html`、`README.md`、`dolls`、`bags` 上傳。
7. 按 `Commit changes`。

## Git 是什麼

Git 是電腦裡用來記錄版本、同步 GitHub 的工具。GitHub 是網站，Git 是你電腦上的同步工具。

如果只用 GitHub 網頁版，可以不用先裝 Git；但每次更新都要手動上傳檔案。

如果要讓 Codex 幫你同步、提交、推送到 GitHub，就需要安裝 Git for Windows。
